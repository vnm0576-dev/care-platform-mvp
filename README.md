# Care Platform MVP

Учебный MVP приложения для связи сиделок, клиентов и пансионатов.

## Цель

Создать приложение, где сиделка может оставить анкету, а клиент может найти сиделку и связаться с ней.

## Учебная цель

Проект используется для изучения:

- GitHub
- Git
- работы с AI-агентами
- структуры репозитория
- разработки мобильного приложения
- подготовки приложения к публикации
