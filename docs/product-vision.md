# Видение продукта

Care Platform MVP — учебный проект приложения для связи сиделок, клиентов и пансионатов.

## Главная идея

Сиделка может создать анкету, а клиент может найти подходящую сиделку и связаться с ней.

## Для кого приложение

- для сиделок, которые ищут работу;
- для клиентов, которым нужен уход за родственником;
- для пансионатов и организаций ухода;
- для администратора, который модерирует анкеты.

## Первый MVP

На первом этапе приложение работает как простая база анкет и контактов.
