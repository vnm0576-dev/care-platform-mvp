# Роли пользователей

## Сиделка

Может зарегистрироваться, заполнить анкету, указать опыт, город, график, навыки и контакты.

## Клиент

Может открыть список сиделок, посмотреть профиль и связаться с подходящим специалистом.

## Администратор

Может просматривать анкеты, скрывать неподходящие профили и модерировать данные.
